# Apex Planner

Apex Planner is a micro-SaaS web application designed to generate highly personalized daily to-do lists and schedules. By taking into account your goals, energy levels, meeting times, and task estimations, it provides an optimized, actionable plan for your day. The focus is on a minimal UI to reduce distractions and boost productivity for freelancers, students, and remote workers.
